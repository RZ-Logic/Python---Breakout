# Python Arcade: Breakout 🧱

A classic, physics-based Breakout game built with Python and Pygame. Features level progression, difficulty scaling, and a mouse-responsive paddle.

## Features
* **Mouse Control:** Ultra-smooth paddle movement responsive to mouse positioning.
* **Level System:** Increasing difficulty with more brick rows as you progress.
* **Physics Logic:** Custom bounce logic that allows players to "aim" the ball based on where it hits the paddle.
* **HUD Display:** Real-time tracking of score, remaining lives, and current level.

## Prerequisites
* Python 3.x
* Pygame library

## Installation & Setup

**1. Clone the repository:**
```bash
git clone https://github.com/rizwanahmed109-beep/Python---Breakout.git
cd Python---Breakout
```

**2. Install dependencies:**
```bash
pip install pygame
```

## How to Run
### Windows
```cmd
python breakout.py
```

### macOS and Linux
```bash
python3 breakout.py
```

## Controls
* **Mouse:** Move left/right to control the paddle.
* **SPACE:** Start game / Next level.
* **R:** Reset game.
